3. Visualizaci√≥n de la carrera de¬†barras

El ‚Äútruco‚Äù para hacer la carrera de barras es utilizar geom_tile en lugar de geom_bar para generar las barra. Esto da un efecto fluido cuando las barras cambian de lugar. Como lo muestra este post de StackOverflow.

            vjust = 0.4, hjust = 0.55, size = 7) +

            vjust = 0.4, hjust = 0.0, size = 7) +
  # nombres

                     '{closest_state}', sep=""),
